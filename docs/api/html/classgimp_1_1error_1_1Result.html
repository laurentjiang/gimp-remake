Detailed Description

template<typename T>
class gimp::error::Result< T >

Result<T> type for error-aware return values.

A Result<T> can hold either a value of type T (success) or an ErrorInfo (failure). This allows functions to return errors without throwing exceptions.

Example usage:

Result<int> Divide(int a, int b) {
if (b == 0) {
return ErrorInfo(ErrorCode::InvalidArgumentOutOfRange, "Division by zero");
}
return a / b;
}
auto result = Divide(10, 2);
if (result.IsOk()) {
std::cout << "Result: " << result.Value() << std::endl;
} else {
std::cerr << "Error: " << result.Error().GetMessage() << std::endl;
}
